5  Propriedades dos estimadores

5.1 Estimadores

Estimadores são estatísticas que assumem valores no espaço amostral. São utilizados para estimar uma função de um parâmetro, seja esse conhecido ou desconhecido. Podemos encontrar situações que a função é o próprio paramentro, sendo assim \(g(\theta) = \theta\).

5.2 Erro Quadrático Médio (MSE)

Em muitas situações, deseja-se comparar se um estimador é melhor que outro. Para isso utilizamos o Erro Quadrático Médio (EQM). O estimador \(T_{1}(X)\) é melhor que o estimador \(T_{2}(X)\) se o erro quadrado médio de \(T_{1}(X)\) for menor que o de \(T_{2}(X)\) para todo \(\theta\).

\[ EQM[T_{1}(X)] < EQM[T_{2}(X)] \]

A definição de estatística é função da amostra. Todo estimador é uma estatística.

Caracteristicas de um bom estimador:

  • Média próxima ou igual a \(g(\theta)\);
  • Menor variância possível.

5.3 Estimador assintoticamente não viciado

Um estimador \(T(X)\) é assintoticamente não viciado para \(g(\theta)\) se o víés (erro) do estimador é igual a 0 quando n tente ao infinito

\[ lim_{n\rightarrow\infty} B(T(X)) = 0, \] ou equivalentemente, se a esperança do estimador for igual a uma função do parâmetro

\[ lim_{n\rightarrow\infty} E[(T(X))] = g(\theta). \] Portanto, ele não possui vício e pode estimar fiélmente o parâmetro se quando n tende ao infinito.

##Estimador Consistente (fracamente)

A principal ferramenta para verificarmos a consistência de um estimador é a desigualdade de chebyshev.

Seja \(X\) uma variável aleatória com média \(\mu\), então: \[ P(|X-\mu| \geq \epsilon) \leq \frac{\sigma^2}{\epsilon^2} \]

Ou seja, a probabilidade do módulo da diferença ser maior ou igual ao erro do estimador é menor ou igual à razão da variância com o erro ao quadrado.

Resultados uteis para provar a consistência de um estimador:

  • O valor esperado da sequência de estimadores \(\{T_{n}\}\) converge para \(\theta\) quando o limite de n tende ao infinito.

\[ lim_{n\rightarrow\infty} E(T_{n}) = \theta \]

  • O variância da sequência de estimadores \(\{T_{n}\}\) converge para 0 quando o limite de n tende ao infinito. (O valor do estimador é um ponto fixo, não varia).

\[ lim_{n\rightarrow\infty} V(T_{n}) = 0 \] Por exemplo, se considerarmos a média amostral \(\overline{X}\), temos que \(E(\overline{X}) = \mu\) e \(V(\overline{X}) = \frac{\sigma^2}{n} \rightarrow 0\), quando \(n \rightarrow \infty\).
