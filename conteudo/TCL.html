3  Teorema Central do Limite

3.1 Função Geradora de Momentos

A partir da função geradora de momentos, podemos derivar os momentos de uma v.a. \(X\), obtendo assim propriedades importantes como média e variância e até a distribuição da variável. Para obtermos esses momentos, temos que avaliar as derivadas da função em torno de \(t = 0\).

\[ m_{x}(t) = E(e^{tX}) = \int_{-∞}^{∞}e^{tx}f_{x}(x)dx \] se X é uma v.a. contínua, e

\[ m_{x}(t) = E(e^{tX}) = \sum_{x}e^{tx}f_{x}(x) \] se X é uma v.a. discreta.

3.1.1 Propriedades

  1. Se \(M_{x}(t)\) existir, então \(M_{x}(t)\) será contínua e diferenciável em uma vizinhança em torno de 0.
  2. Os momentos de ordem \(r\) de \(f_{x}(·)\) podem ser obtidos da seguinte forma \[\frac{d^r}{dt^r} m_{x}(0) = E(X^r) = \mu_{r}.\]
  3. Se \(M_{x}(t) = M_{y}(t)\), então: \[ X\overset{\mathrm{iid}}{\sim}Y.\] Portanto, a função geradora de momentos determina a distribuição de uma v.a. X.
  4. A FGM da soma de variáveis aleatórias independentes é o produto das FGM de cada uma delas: \[M_{x+y}(t) = M_{x}(t)M_{y}(t).\]

3.2 Transformada de Fourier

Utilizamos a transformada de Fourier para achar uma função que se aproxime da função de distribuição da variável \(X\). Essa transformada expressa uma função em termos de funções de base sinusoidal, sendo ela A partir do segundo momento amostral conseguimos a variância de \(X\). 3.3 Teorema Central do Limite

Seja \(\{X_{i}\}_{i \geq 1}\) uma sequencia de v.a’s iid, \(X_{i}\) com média \(\mu\) e variância $^2 $ finitas. Sendo \(S_{n}\) a soma dessas v.a.’s, então a distribuição \(z_{n} = \frac{S_{n} - n\mu}{\sigma \sqrt{n}}\) converge em distribuição para uma \(N(0,1)\) quando o n é suficientemente grande. Esse teorema nos permite utilizar a normal como base em amostras grandes que não possuem distribuição especifica.
